Detailed Description

This library is included in the Zumo Shield libraries as a dependency for the ZumoReflectanceSensorArray library. If you are programming an Arduino on a Zumo, you should generally use the ZumoReflectanceSensorArray library instead of using the QTRSensors library directly.

For documentation specific to the QTRSensors library, please see the user's guide on Pololu's website.
